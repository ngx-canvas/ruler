# @ngx-canvas/ruler

This is a Typescript Library that makes canvas intergrations easier!

## Installation

```sh
$ npm i --save @ngx-canvas/ruler
```

## Usage

Add package to NgModule imports:

```js
import { RulerModule } from '@ngx-canvas/ruler';

@NgModule({
    ...
    imports: [
        RulerModule
    ]
    ...
})
```

Add component to your page:
```html
<ruler>
    ...
</ruler>
```
